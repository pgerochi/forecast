## Install
Clone the repo via git:

```bash
git clone https://github.com/pgerochi/forecast.git
```

And then install dependencies.

```bash
$ cd 
$ npm start
```

## Run Application - devolopment mode

```bash
$ npm run dev
```

## Packaging

To package apps for the local platform:

```bash
$ npm run package
```
